# Flippers - Online Random Decision Maker

Flippers is a suite of free, interactive, and beautifully designed online decision-making tools. Whether you need to flip a coin, roll some dice, shuffle cards, or spin a wheel, Flippers provides a seamless, fast, and secure experience. 

The entire project is built with a modern **Glassmorphism** design system, featuring frosted glass UI elements, ambient floating background animations, and full dark/light mode support.

## 🛠️ Tools Included

- **[Coin Flipper](coin_flipper_glassmorphism.html)**: Instant heads or tails with smooth coin toss animations.
- **[Dice Roller](dice_roller_glassmorphism.html)**: Roll 1 to 6 virtual dice instantly. Perfect for board games and RPGs.
- **[Wheel Spinner](wheel_spinner_glassmorphism.html)**: Create a custom spinning wheel with unlimited options to randomly pick winners.
- **[Card Shuffler](card_shuffle_glassmorphism.html)**: Shuffle and draw random playing cards instantly from a standard 52-card deck.
- **[Decision Shuffler](decision_shuffler_glassmorphism.html)**: Enter custom options and let the tool shuffle and pick one for you.
- **[Comment Picker](comment_picker_glassmorphism.html)**: Quickly pick random winners from a list of inputted text or comments.
- **[Random Number Generator](number_generator_glassmorphism.html)**: Generate truly random numbers within a custom minimum and maximum range.
- **[Fantasy Name Generator](fantasy_name_glassmorphism.html)**: Instantly generate magical names for your RPG characters, worlds, and stories.

## ✨ Key Features

- **Glassmorphism UI**: Beautiful, premium frosted-glass interface that stands out.
- **Dark/Light Theme**: Fully integrated theme toggle with `localStorage` memory to save user preferences.
- **Responsive Design**: Flawless experience on both desktop and mobile devices.
- **No Dependencies**: Built entirely with Vanilla HTML, CSS, and JavaScript. No bulky frameworks (like Tailwind or React) required for the final glass versions.
- **Fast & Interactive**: Smooth CSS animations and instant JavaScript logic for a highly responsive feel.

## 🚀 Getting Started

Since Flippers is built with plain HTML, CSS, and JS, there is no build step or package manager required! 

1. Clone or download the repository to your local machine.
2. Open `index.html` (or `flippers_glassmorphism.html`) in any modern web browser.
3. Start spinning, picking, and deciding!

## 🎨 Design System Note

The project was recently migrated from Tailwind CSS to a custom Vanilla CSS **Glassmorphism** architecture to provide a more premium aesthetic. All files ending in `_glassmorphism.html` represent the finalized, highly polished versions of the tools. They share a unified CSS structure utilizing CSS Variables for easy maintenance.
